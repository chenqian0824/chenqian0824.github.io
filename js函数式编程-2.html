组合的特性

1. 结合律（associativity）    


**其实只要把握好（从右向左执行的顺序即可）**
2. pointfree模式
定义：永远不必说出你的数据，函数无须提及将要操作的数据是什么样的。

debug



// id 的实用函数
var id = function (x) { return x };
// 结论成立
compose(id, f) === compose(f, id) === f;
总结：组合像一系列管道那样把不同的函数联系在一起，数据就可以也必须在其中流动——毕竟纯函数就是输入对输出
